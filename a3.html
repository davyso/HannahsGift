  • Move It! Move It!

    Start game by selecting/clicking how many cards you would like to play. Click any card that you would like to start with and the game will begin

    Use card directions to select next card -ex. "2 up and 3 right"
    New directions are given for wrong selection verbally -ex. "Oops, you need to be 1 up and 1 right"

    A sound will play after each correctly answered question. When board is cleared the board will show a picture

    You have the option to return to the main screen by selecting/clicking the "Back" button, start a different game by selecting/clicking the "New game" button and just keep the same card count by the "Restart" button.

  • For True Fans Only

    On main screen you will need to select/click a theme either Winnie the Pooh, Percy Jackson or Harry Potter

    The video will start and after a few second will ask a question. After the question has been answered then the video will start again or go back to main screen

    When finished you may go back to main title screen by selecting/clicking the "Back to Main Screen" button.

  • Can You Hear Me Now?

    Once song starts playing you must use the arrow buttons on the sides to rotate the middle circle to find where the source has centered and not louder on one side or the other

    After you have stopped where you think the source it centered then you select/click the middle arrow button in the center of the circle
    You will be given a 100 points per song if selected right the first time. You will loose points for each time the middle arrow is selected for the wrong source

    You may go back to the main screen by selecting/clicking the "Home" button at the top left

What is Hannah's Gift?

3-1 game system (PHP, MySQL, phpMyadmin, JS, jQuery, HTML, Bootstrap, WebAudio, and CSS) Direction card game, Audio game, Video pop-up game. The target audience is people with disabilities that need or can use EyeGaze or a mouse (one click). Main page will have the option to play any of the three games. Direction card game will have pictures. The Audio game has song that you will need to find the source from. Video Pop-Up game will have option to play three different themes (Winnie the Pooh, Percy Jackson and Harry Potter).
